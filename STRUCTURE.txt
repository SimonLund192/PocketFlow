API endpoints (from backend/app/routes and backend/app/main.py)

Core
- GET /
- GET /health

Auth (prefix /auth)
- POST /auth/register
- POST /auth/login
- POST /auth/logout
- GET /auth/me
- PATCH /auth/me

Categories (prefix /api/categories)
- POST /api/categories/
- GET /api/categories/
- GET /api/categories/{category_id}
- PUT /api/categories/{category_id}
- DELETE /api/categories/{category_id}

Budgets (prefix /api/budgets)
- POST /api/budgets/
- GET /api/budgets/
- GET /api/budgets/{budget_id}
- GET /api/budgets/by-month/{month}
- PUT /api/budgets/{budget_id}
- DELETE /api/budgets/{budget_id}

Budget Line Items (prefix /api/budget-line-items)
- POST /api/budget-line-items/
- GET /api/budget-line-items/
- GET /api/budget-line-items/{line_item_id}
- GET /api/budget-line-items/budget/{budget_id}
- PUT /api/budget-line-items/{line_item_id}
- DELETE /api/budget-line-items/{line_item_id}

Goals (prefix /api/goals)
- POST /api/goals
- GET /api/goals
- PUT /api/goals/reorder
- PUT /api/goals/{goal_id}
- DELETE /api/goals/{goal_id}

Transactions (prefix /api/transactions)
- GET /api/transactions
- POST /api/transactions
- DELETE /api/transactions/{transaction_id}

Dashboard (prefix /api/dashboard)
- GET /api/dashboard/stats
- GET /api/dashboard/balance-trends
- GET /api/dashboard/expense-breakdown

Database (prefix /api/database)
- GET /api/database/collections
- GET /api/database/{collection_name}
- GET /api/database/{collection_name}/count

AI (prefix /api/ai)
- POST /api/ai/chat

Admin (prefix /api/admin)
- DELETE /api/admin/clear-transactions
- DELETE /api/admin/clear-budgets
- DELETE /api/admin/clear-categories
- DELETE /api/admin/clear-goals
- DELETE /api/admin/clear-all
